Columns and Margins

The grid is formatted to allow the grid columns to float inline with each other. Wrap an inline grid with a class of .row. Then add whatever columns you want to make sixteen. The base spacing between each column is 2%. In some cases, like a three column situation, the numbers won't add up to sixteen. In this case, give the second and third column divs a class of .alt-spacing

Typography

Build a hierarchy of information through the use of standard typography. Users will navigate content this way by scanning for large headlines and big type. All font sizing is set to rem with fallbacks to ems and px. Icons can also be helpful in helping communicate a particular action to a user. Wirefy comes with Font Awesome ready for your use.

Buttons

Buttons are intended for action and thus should have appropriate weight. The standard button is given that weight with a little bit of depth and a strong hover. You can either indicate a button element or add a class of .button to any a element.

Media Queries

For a more solid foundation across a larger sea of devices, we have opted to use ems as our media query baseline instead of set pixel screens. This way our structure stays proportional and fluid since we can't control how a user will view our site or what new devices displays will come to life in the future. For example, if zooming is a common practice for your audience then this will help keep your foundation intact the way you intended it.
