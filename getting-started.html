Download

Before you download the latest version of Wirefy, make sure you have a code editor available. We recommend TextMate or Coda2. You will also need some working knowledge of HTML and CSS. Ok now that we have that settled let's get started. We've strived to keep Wirefy as compact as possible and continue to work on techniques to better our compression. Currently the entire document is just under ~700Kb.

File Structure

With the downloaded folder, you'll find the following file structure, contents and logically grouping common assets. You'll get everything from JavaScript files to the CSS and startup placeholder images. Includes are both compiled and minified versions of most JavaScript files. Wirefy has been built in a way to get your wireframes up and running faster. Please note that all JavaScript plugins require jQuery to be included.

HTML Template

Now that you are familiar with the file structure of Wirefy, we can focus on getting you up and running. We'll open the HTML file included in our download. Let's take a look at what you will see when you open this file. We don't want you to waste time adding CSS files or JavaScript files. These have all been included for you out of the gate.

Extend

We've shown you what is included in Wirefy and how to easy it is to get started. But sometimes you want to do more than what has been included. Wirefy is flexible enough to work with other frameworks. You can add additional JS files to the js folder included then just call them into your basic HTML file. Please note that if you add a new framework that we haven't tested we won't be able to offer proper support if you run into issues.
